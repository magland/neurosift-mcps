# Neurosift MCP Servers

MCP servers for neuroscience research tools including DANDI archive search, NWB file analysis, OpenNeuro integration, and scientific plot analysis.

## Installation

### Add the marketplace
```
/plugin marketplace add https://github.com/magland/neurosift-mcps.git
```

### Install MCP servers
```
/plugin install neurosift-tools@neurosift-mcps
/plugin install plot-vision@neurosift-mcps
```
